### Update Contact Information

1. Open `contact.html` in a text editor
2. Find the contact details section
3. Update phone, email, and address

### Add Google Maps

1. Go to Google Maps
2. Search for your academy location
3. Click "Share" â†’ "Embed a map"
4. Copy the iframe code

### Update Coach Information

1. Open `coaches.html` in a text editor
2. Find the coach cards section
3. Update coach names, roles, and achievements

### Change Colors

1. Open `styles.css`
2. Find the `:root` section at the top
3. Modify the color variables:
   - `--gold-primary` - Main gold color
   - `--gold-dark` - Darker gold
   - `--silver-primary` - Main silver color
   - `--silver-dark` - Darker silver
